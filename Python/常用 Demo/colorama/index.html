    colorama¶

    概述¶

    colorama module 用于打印不同颜色的文本。

    用法示例¶

    import colorama

colorama.init()
print(colorama.Fore.RED + 'This is red')

    from colorama import *

init()
print(Fore.BLUE + 'This is blue')

    from colorama import init, Fore

init()
print(Fore.GREEN + 'This is green')

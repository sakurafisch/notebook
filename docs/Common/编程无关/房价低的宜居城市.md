# 房价低的宜居城市

- 在资源有限的情况下，尽量不考虑云南、新疆、西藏，以及东北。因为云南、新疆、西藏不能用无通话功能的纯流量卡上网，东北太冷且商业环境相对来说不够开放。
- 在东北之中，辽宁比吉林、黑龙江在经济上更有优势、地理上更近北京，而且在国防方面部署更好。
- 为了更易于融入当地文化，尽量选择汉族聚居地而不选择少数民族聚居地。
- 功能重叠的居住地不重复考虑。

## 海景房且海鲜便宜的

- 山东省龙口市

- 山东省威海市乳山市

# 工具链维护

## Arch Linux

全局更新
```bash
sudo pacman -Syyu
sudo yay -Syyu
rustup update
```

更新`nvm`
```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.4/install.sh | bash
```


更新`npm`
```cmd
npm install -g npm
```

安装最新版本`Node`及其对应的`npm`
```bash
nvm install --lts
nvm install-latest-npm
```

更新`pip`
```
python -m pip install --upgrade pip
```

查看`JDK`版本
```bash
archlinux-java status
```

## Windows

更新`Visual Studio` 现有模块
```cmd
"C:\Program Files (x86)\Microsoft Visual Studio\Installer\setup.exe" updateAll
```

更新`rust`
```cmd
rustup update
```

更新`pip`
```cmd
python -m pip install --upgrade pip
```

看看有没有新的`Python`要安装
```cmd
pyenv versions
pyenv install -l
```

启动`Windows更新`并安装
```cmd
UsoClient ScanInstallWait
```

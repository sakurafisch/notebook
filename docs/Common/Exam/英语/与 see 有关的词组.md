# 与 see 有关的词组

- see things   发生幻觉；见神见鬼
- see stars   眼冒金星
- see a man about a dog   有点急事，先行告退
- see the elephant   见见世面
- see the lions   游览名胜古迹
- see with half an eye   一看就明白
- see the back of sb.   看（某人）离开，赶走 [摆脱]（某人）
- see the back of sb/sth    再也不愿见到某人
- see eye to eye   观点一致
- see through 看透
- see 做名词，表 “主教教区、教廷”

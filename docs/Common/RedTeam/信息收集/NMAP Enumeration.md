# NMAP Enumeration

## NMAP Scan Types

```
# TCP Scan
nmap -sT

# UDP Scan
nmap -sU

# Stealth Scan
nmap -sS
```

## NMAP Basics

```
# Commands go here
nmap -sC -sV $TARGET
```

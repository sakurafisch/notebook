- start from scratch	从头开始
- for simplicity’s sake	为了简单起见
- within this directory	在这个目录中
- As a last resort	最后的办法；最终手段
- parallel	平行的
- elementary	初级的
- flesh out	使充实，使丰满
- a significant amount of 	大量
- hierarchy		等级制度；阶层
- terminology	术语
- validate	验证
- applets	小程序
- retrieve	检索
- standalone	n.脱机	adj.单独的，独立的

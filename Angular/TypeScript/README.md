# TypeScript

## 语法特性

- 类 Classes
- 接口 Interfaces
- 模块 Modules 
- 类型注解 Type annotations
- 编译时类型检查 Compile time type checking 
- Arrow 函数 (类似 C# 的 Lambda 表达式)

## 安装和编译

```bash
$ npm install -g typescript # 安装
$ tsc filename.ts # 编译
```

默认情况下编译器以ECMAScript 3（ES3）为目标但ES5也是受支持的一个选项。
